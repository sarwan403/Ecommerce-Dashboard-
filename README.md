# Ecommerce-Dashboard-
📊 Power BI Sales Analysis Dashboard
📌 Project Overview
This project focuses on transforming raw sales data from an Excel dataset into an interactive and insightful Power BI dashboard. The goal is to perform data cleaning, transformation, and visualization to uncover meaningful business insights.
The dataset contains transactional sales records including order details, shipping information, customer IDs, product IDs, quantities, and discount values.

🗂️ Dataset Description
The original dataset was provided in Excel format and includes the following key fields:

Order Information: Order ID, Order Date, Ship Date, Delivery Date
Shipping Details: Ship Mode (First Class, Second Class, Standard Class, Same Day)
Customer & Product Data: Customer ID, Product ID
Sales Metrics: Quantity, Discount


🔄 Data Transformation (Power BI)
The following transformations were applied in Power BI:

✅ Converted date fields into proper date format
✅ Created calculated columns (e.g., delivery duration)
✅ Handled missing or inconsistent values
✅ Standardized categorical fields (Ship Mode, Product categories)
✅ Enabled data modeling for efficient relationships


📈 Dashboard & Visualizations
The Power BI dashboard contains multiple interactive charts:
🔹 Key Visuals


Sales Trend Analysis

Orders over time (Year-wise / Monthly trends)



Ship Mode Analysis

Comparison of shipment types (Standard, First Class, Same Day)



Discount Impact Analysis

Effect of discounts on quantity sold



Product-Level Insights

High-performing vs low-performing products



Customer Behavior Insights

Order frequency and purchasing patterns




📊 Key Insights

📌 Standard Class shipping is most commonly used
📌 Discounts significantly influence bulk purchases
📌 Certain products have consistently higher demand
📌 Seasonal variation observed in order trends


🛠️ Tools & Technologies Used

Microsoft Excel – Data source
Power BI Desktop – Data transformation & visualization
DAX – Calculated measures and columns

Download the .pbix Power BI file from this repository
Open using Power BI Desktop
Explore the dashboard using filters and slicers
